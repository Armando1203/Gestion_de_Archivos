# Gestion_de_Archivos
Un gestor de archivos de imagen y de texto, para los archivos de imagen 
solamente se manejan las extensiones jpg, jpeg, png y gif, para los 
archivos de texto solo es en extension txt.

El programa abre un archivo con cualquiera de las extensiones antes
mencionadas, si detecta una extension correcta, abrirá el archivo en el
panel correspondiente, después podrá guardar nuevamente el archivo con
otra extensión, con la misma o incluso en los archivos de texto,
podrá editar el texto escrito.

Lenguaje de Programación: Java
Programa: Apache Netbeans Versión 17 + JDK 17




# File_Management
An image and text file manager, for image files
only the jpg, jpeg, png and gif extensions are handled, for
text files only have a txt extension.

The program opens a file with any of the extensions before
mentioned, if it detects a correct extension, it will open the file in the
corresponding panel, then you can save the file again with
another extension, with the same or even in the text files,
You can edit the written text.

Programming Language: Java
Program: Apache Netbeans Version 17 + JDK 17
